# Zen Class MongoDB Task

## Database Name
zen_class

## Collections
users, mentors, topics, tasks, attendance, codekata, company_drives, placements, task_submissions

## Queries Implemented
1. Topics and tasks taught in October
2. Company drives between 15-Oct-2020 and 31-Oct-2020
3. Company drives and students appeared for placement
4. Problems solved by user in codekata
5. Mentors with mentee count > 15
6. Absent users and task not submitted between 15-Oct and 31-Oct
